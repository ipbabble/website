## Which Tool Should I Use?

### For Interactive Development (Recommended)

**Cursor IDE** - Primary Tool

- **Best for**: Full-stack development with AI pair programming
- **Strengths**: Codebase awareness, inline editing, MCP integration
- **Use when**: Building applications, writing code, developing features
- **Status**: Fully supported with complete documentation
- **[Get Started with Cursor](cursor)**

### For Conversational Workflows

**Claude Desktop** - Experimental

- **Best for**: Conversational analysis and MCP-powered workflows
- **Strengths**: Natural conversations, powerful reasoning, MCP protocol
- **Use when**: Data exploration, documentation, complex queries
- **Status**: Configuration documented
- **[Get Started with Claude Desktop](claude-desktop)**

## Common Features

All recommended tools support:

- **MCP Integration** - Connect to MCP servers
- **Code Generation** - AI-powered development  
- **Natural Language** - Describe what you want to build
- **Context Awareness** - Understands your codebase

## Getting Started Checklist

Before using any AI tool:

- [ ] **Tool Installation** - Download and install your chosen tool
- [ ] **MCP Server Setup** - Deploy or connect to MCP servers
- [ ] **Configuration** - Set up MCP integration
- [ ] **Testing** - Verify connection and functionality
